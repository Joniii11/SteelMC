Expand description

Column-level cache for flat-cached (xz-only) density function results.

Call ensure before reading values. Values are recomputed only when (x, z) changes.

pub fn ensure(&mut self, x: i32, z: i32, noises: &OverworldNoises)

Ensure the cache is populated for the given (x, z) block coordinates.

If the cache already holds values for this column, this is a no-op. Computes all flat-cached density functions in topological order, then any Y-independent router entries.
