Density function types matching vanilla Minecraftâ€™s DensityFunctions.java

Each density function type is its own struct, mirroring vanillaâ€™s separate record/class pattern. The DensityFunction enum wraps them for dispatch.

These types are used at build time by the density function transpiler to parse JSON density function trees and generate native Rust code. Runtime evaluation is done by the transpiled code, not by interpreting this tree.
